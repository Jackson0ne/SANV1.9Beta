<h1 align="center">Welcome to Steam Achievement Notifier (V1.9) Beta!</h1>

To get started, please read the information below, then navigate to the [Beta Releases page](https://github.com/Jackson0ne/SANV1.9Beta/releases) to download the latest available **Beta** version of **Steam Achievement Notifier (V1.9)**.

> [!WARNING]
> ⚠️ After the latest official version has been released, all **Beta versions** will stop working. You will need to manually download and install the latest public release `EXE`/`AppImage` in order to continue using **Steam Achievement Notifier** from that point.
> 
> When released, the official `EXEs`/`AppImages` will be available to download on the [Steam Achievement Notifier Releases page](https://github.com/SteamAchievementNotifier/SteamAchievementNotifier/releases), or linked in the `#⁠updates` channel on [Discord](https://discord.com/invite/FxCFtpd3eu).

<h2>Installation & Backup</h2>

> [!NOTE]
> To install (or to install new revisions once released):
> - Download the latest `EXE`/`AppImage` from the [Beta Releases page](https://github.com/Jackson0ne/SANV1.9Beta/releases)
> - Run the installer
> 
> This will install the **Beta version** of **Steam Achievement Notifier (V1.9)** over the top of your existing installation.
>
> ⚠️ *Please see below if you wish to back up your existing app config before testing out any available **Beta versions**.*

> [!TIP]
> If you wish to back up your existing app config before testing out any available **Beta versions**, please do the following:
>
> - Open the Start menu, type `%localappdata%` and press Enter
>   - :penguin: Or on Linux, navigate to `~/.local/share`
>
> - In the folder that appears, find `Steam Achievement Notifier (V1.9)`
> - Copy this folder somewhere safe
>
> In the event you wish to restore this backup:
>
> - Open the Start menu, type `%localappdata%` and press Enter
>   - :penguin: Or on Linux, navigate to `~/.local/share`
> - In the folder that appears, find `Steam Achievement Notifier (V1.9)` and either delete it, or rename it (e.g. `Steam Achievement Notifier (V1.9).old`)
> - Copy your `Steam Achievement Notifier (V1.9)` backup folder (from somewhere safe) and paste it back into the original location

> ⚠️ *If your original app config is still not being loaded, please ensure the folder is named `Steam Achievement Notifier (V1.9)` *exactly*. Otherwise, it will not get picked up by **Steam Achievement Notifier** on launch, and a default folder will be recreated and used instead.*
